# Mesto

Проектная работа «Mesto. Валидация форм и сборка кода».

## Что реализовано

- Подключён Vite.
- Добавлены команды `npm run dev`, `npm run build`, `npm run preview`.
- Создан файл `vite.config.js`, dev-сервер автоматически открывает проект в браузере.
- В формах отключена встроенная браузерная валидация через `novalidate`.
- Кнопки форм изначально неактивны: добавлены `popup__button_disabled` и `disabled`.
- В HTML добавлены атрибуты `required`, `minlength`, `maxlength`, `pattern`, `type="url"`.
- После каждого поля добавлены элементы для вывода ошибок.
- Валидация вынесена в универсальный модуль `src/scripts/components/validation.js`.
- Карточки создаются из массива `initialCards`.
- Работают лайки, удаление карточек, просмотр картинки, редактирование профиля, добавление карточки и изменение аватара.

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

После сборки появится папка `dist`.
